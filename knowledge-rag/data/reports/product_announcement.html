Acme launches Knowledge RAG 2.4

On 2026-04-15 Acme Inc. released Knowledge RAG 2.4, a private agentic retrieval-augmented generation system. The release follows six months of refactoring that decoupled the system from any specific subject domain.

What's new

The 2.4 release ships multi-format ingestion (.docx, .md, .txt, .pdf, .html), a YAML-based domain-pack mechanism, and a real streaming UI. Customers in HR, legal, and finance have already onboarded their own domain packs against the same code base.
